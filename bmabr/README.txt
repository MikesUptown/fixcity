

Overview for fxicity.org and build me a bike rack. 
================================================== 


This is a (geo)django based application that allows citizens to
request bike racks in the community. It makes extensive use of
PostGIS, Geos, and OpenLayers. For more information on this mapping
libraries please see their

--------------- 
This guide will help you get up and running with fixcity 


Database help 
+++++++++++++++

You must have PostGIS install on your system. On ubuntu/debian systems
this can be accomplished with the command:: 
     apt-get install postgis 

A similar process can be followed for redhat using yum.
